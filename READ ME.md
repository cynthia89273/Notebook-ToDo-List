# Notebook To-Do List

## Overview

Notebook To-Do List is a simple web application that helps users keep track of their daily tasks. The application is designed to resemble a traditional notebook page, featuring blue horizontal lines, a red margin line, and a clean paper-like layout on a dark background.

Users can add tasks, mark completed tasks with a strikethrough effect, and keep their tasks saved even after refreshing the page through the use of browser local storage.

## Features

* Add new tasks
* Mark tasks as completed by clicking on them
* Completed tasks display a strikethrough effect
* Tasks are automatically saved using local storage
* Tasks remain available after refreshing the page
* Notebook-inspired user interface
* Responsive and lightweight design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Browser Local Storage

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Start adding and managing your tasks.

No additional installation or dependencies are required.

## How to Use

1. Enter a task in the input field.
2. Click the **Add** button.
3. The task will appear on the notebook page.
4. Click a task to mark it as completed.
5. Click the task again if you want to remove the completed status.
6. Refreshing the page will not remove saved tasks.
